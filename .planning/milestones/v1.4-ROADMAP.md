# Milestone v1.4: Installation & Build Guide

**Status:** SHIPPED 2026-02-17

## Overview

Bulletproof installation instructions for both end users (pip install) and contributors (build from source) on Cygwin/Windows and Linux, with a verification script to confirm the setup works.

**Depends on**: Nothing (documentation-only, no code dependencies)
**Requirements**: INST-01, INST-02, INST-03, INST-04, INST-05
**Plans**: 2 plans
